# Github에 파일을 올리기 위한 명령어
## 1. git status
#### 현재 폴더의 git 상태를 확인할 수 있다.
#### add 된 파일이나 remote repository와 다른 부분을 알 수 있다.
<br>

## 2. git add 파일명
#### 파일을 add 한다.
<br>

## 3. git commit -m "메모"
#### add 했던 파일을 메모와 함께 commit 한다.
<br>

## 4. git push origin main
#### commit 했던 파일들을 github의 remote repository에 push한다.
